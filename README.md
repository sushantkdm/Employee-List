# Adding-Employees
This is a simple web application that allows you to add and delete employees. It is built using HTML, CSS, and JavaScript.

Usage:

- To use this application, simply open the index.html file in your web browser. You will see a form where you can add new employees by filling out their name, profession, and age, and clicking the "Add Employee" button. Once you have added some employees, you will see them displayed in a list below the form. Each employee in the list has a "Delete" button that allows you to remove that employee from the list.

Features:

- Add new employees by filling out their name, profession, and age.
- Delete employees from the list by clicking the "Delete" button next to their name.
